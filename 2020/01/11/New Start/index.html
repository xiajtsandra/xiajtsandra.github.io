New Start——一个新的开始

发表于 2020-01-11 更新于 2020-01-12 阅读次数： 阅读次数： Valine：

首先，
欢迎来到夏小姐的blog！
这是一个现在还在完善中的、菜菜的、但会不断成长的blog。

Brief description

Origin

在2020年1月12日，夏小姐花了一个下午加一个晚上的时间搭建好了这个小空间。搭建这个小空间的想法出现是因为这段时日在搜索学习的过程中遇到了很多大佬，在他们的blog里获得了很多帮助，也有了一些感触。接下来的日子里，希望可以在这个空间，记录下自己的收获，找到放向。

Process

搭建这个网站还是花了夏小姐一些脑细胞的。毕竟，因为夏小姐看到英文头疼，所以夏小姐看到代码也头疼(bushi)。但因为有意思，所以讨厌看代码的夏小姐看了一下午+一晚上的代码。T_T"
BY the way， 好歹整出来了不是（我真是个小天才！！），虽然还有很多很多bug，但夏小姐再看下去就要死掉了。

好了说正事，凭空搭出blog对我来说是不可能的，这里我选用了hexo作为架构，然后next作为美化（不得不说，next挺好看的,右下角更好看！)

Finally

这可能算是我第一次利用代码自己干了一些事情，还是很有成就的。这片空间我会好好维护，不定期更新。虽然不像那些业界大神一样样样精通，但会记录一些成长，也希望这个blog的搭建也能在一定程度上激励我更加的努力，成为想成为的人。放下一些过去，忘掉一些不愉快的曾经，作为2020——也是人生一个新的开始，不迷茫。
